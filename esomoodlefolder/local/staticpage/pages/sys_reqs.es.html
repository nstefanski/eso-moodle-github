Requisitos del Sistema

Conexión a Internet: Cable o DSL
Sistema operativo: XP, Vista, Windows 7, Mac OSX o mayor
Procesador del ordenador: CPU con soporte SSE2, incluyendo Intel Pentium 4, Pentium M, Core o Atom, AMD Athlon 64 o posterior
Memoria de la Computadora: 512 MB RAM o superior
Resolución de pantalla: 1024 x 768 mínimo

! El navegador que recomendamos es Google Chrome, y si usted está experimentando dificultades técnicas no relacionadas con el contenido del curso, le pediremos que descarge este navegador. Sin embargo, los navegadores como Mozilla Firefox, Internet Explorer o Safari también trabajarán con los ajustes actualizados.
